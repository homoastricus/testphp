4. Веб-сервер Nginx
Документация на английском

Основные команды (актуально для ос типа debian / ubuntu):
service nginx restart - перегрузка,
service nginx stop - остановка веб-сервера,
service nginx start - запуск веб-сервера,
nginx -t - проверка валидациии конфигурации сервера и статус
service nginx status - статус (информация о состоянии) веб-сервера
cd /etc/nginx - расположение конфигурации
cd /etc/nginx/site-available - расположение конфигурации виртуальных хостов

5*. Модуль Рhp-fpm
данные команды актуальны непосредственно для учебного сервера, на других серверах, на других ОС и для других версий интерпретатора php команда может существенно отличаться*)

service php8.2-fpm restart - Перегрузка модуля php-fpm
service php8.2-fpm start - запуск модуля php-fpm
service php8.2-fpm stop - остановка модуля php-fpm
service php8.2-fpm status - вывод статуса активности модуля php-fpm
